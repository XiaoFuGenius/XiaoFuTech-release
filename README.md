# XiaoFuTech-release
iOS 快捷开发工具包 XiaoFuTech.framework for Release. https://www.xiaofutech.com

## 使用
- Podfile中指定仓库位置，

```
# Master仓库不要遗漏
source 'https://github.com/CocoaPods/Specs.git'
source 'https://github.com/XiaoFuGenius/xiaofu-specs.git'
```

- 为工程target添加依赖，

```
pod 'XiaoFuTech-release'
```
